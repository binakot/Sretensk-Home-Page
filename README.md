## Setup

1. Install [Node.js](https://nodejs.org) and [NPM](https://docs.npmjs.com/getting-started/installing-node)

2. Install [Gulp](https://gulpjs.com)

3. Run `npm i`

## Development

* Run `npm run serve`

## Build

* Run `npm run build`
